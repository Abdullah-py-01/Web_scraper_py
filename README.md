# Web_scraper_py
A Python web scraper that automatically extracts book titles, prices and ratings from websites and saves them to a clean Excel file.
# Web Scraper - Books Data Extractor

## What It Does
This Python script automatically scrapes 
book data from websites and saves it to 
a clean Excel file in seconds.

No more manually copying data from websites 
— just run the script and get all the data 
you need instantly.

## Features
- Scrapes 1000+ books automatically
- Extracts title, price and rating
- Loops through 50 pages automatically
- Cleans price formatting
- Saves clean data to Excel
- Shows progress while scraping

## Tools Used
- Python
- Requests
- BeautifulSoup4
- Pandas
- OpenPyXL

## How To Run

1. Install required libraries:
pip install requests beautifulsoup4 
pandas openpyxl

2. Run the script:
python web_scraper.py

3. Check books.xlsx for results

## Sample Output
- Total books scraped: 1000
- Data includes: Title, Price, Rating
- Output saved to: books.xlsx
